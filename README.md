# Anice Dusk
Dark OSX themed look for Aseprite
Currently in development

## Installation:
Put theme folder into `extensions` folder inside your Aseprite preferences folder.

**Windows:**
```
  %AppData%\Aseprite
```

**macOS:**
```
  ~/Library/Application Support/Aseprite
```

**Linux:**
```
  ~/.config/aseprite
```
